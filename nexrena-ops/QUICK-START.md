## 30-Second Summary

**New client:** Lead -> Proposal -> Project -> Invoice -> Paid

**Repeat client:** Project -> Invoice -> Paid

**Monthly billing:** Dup last invoice -> Update date -> Send -> Mark paid

Everything auto-fills from previous steps. Start at `/leads` or `/crm`.
