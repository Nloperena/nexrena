# Nexrena Ops — Subscriptions & Recurring Services

Covered in the **Monthly / Recurring Billing shortcut** section of **[WORKFLOW.md](./WORKFLOW.md)**.

**TL;DR:** Same client, same service, every month.

1. Create the project once (status: `execution`, don't mark as delivered)
2. Create the first invoice manually
3. Every month after: **Dup** last invoice -> update date -> send -> mark paid

Link expenses to the project if you want profitability tracking in `/reports`.
