# Nexrena Ops — Repeat Clients

Covered in the **Repeat Client shortcut** section of **[WORKFLOW.md](./WORKFLOW.md)**.

**TL;DR:** Client already in CRM. Skip leads and proposals.

1. `/projects` -> **+ New Project** -> select existing client
2. `/invoices` -> **+ New** -> select client (everything auto-fills)
3. Click **Paid** when payment arrives

For monthly retainers, use **Dup** on the last invoice instead of creating from scratch.
