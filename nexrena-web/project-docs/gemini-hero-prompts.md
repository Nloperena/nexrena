# Gemini Hero SVG Prompt Packs

This document contains ready-to-run prompts to generate meaningful, section-specific animated SVG headers using Gemini. Each prompt is designed to produce a production-ready SVG and CSS animation that follows your brand constraints (Obsidian and Gold palette, subtle app-like motion, and semantic meaning).

---

## Global Instructions (To prefix any prompt if needed)
*Always ensure the SVG operates smoothly on the dark obsidian background (`#0C0F12`) using gold accents (`#C9A96E`, `#B89354`, `#F5D796`, and `rgba()` variations). Ensure `prefers-reduced-motion` is included.*

---

## 1) Home Section Prompt

```text
You are a senior motion systems designer and front-end SVG engineer.
Generate production-ready inline SVG + CSS animation for the HOME hero header.

Message to communicate:
- "Nexrena builds growth infrastructure, not one-off marketing tactics."
- Visual Metaphor: Strategic momentum, a core engine powering outward growth.

Requirements:
- Mood: Confident, technical, powerful yet restrained.
- Visuals: Think along the lines of a central 'core' or 'engine' that seamlessly distributes structured lines of data/energy outward, symbolizing robust infrastructure.
- Brand constraints: Obsidian background (#0C0F12), restrained gold highlights (#C9A96E, #B89354).
- Motion: Subtle pulsing and continuous, smooth directional flow (not glitchy).
- Output: 
  1. The raw inline `<svg>` code (viewBox="0 0 1200 675").
  2. The scoped CSS `<style>` block for animations.
  3. Include `@media (prefers-reduced-motion: reduce)` to pause animations.
  4. Ensure it acts as a background without obstructing foreground text.
```
