Launch isn't the end. It's the start of monitoring. Here's the checklist.

## Verify

- **Redirects** — Every old URL redirects. Test a sample. Use a crawler.
- **Analytics** — Events fire. Conversions track. No broken tracking.
- **Forms** — Submit forms. Emails arrive. No broken submissions.
- **Links** — Internal links work. No 404s. Check critical paths.

## Search

- **Sitemap** — Submitted. New URLs.
- **Indexation** — Check Search Console. Pages being indexed?
- **Core Web Vitals** — Still passing? No regression.

## Monitor

- **First week** — Traffic, errors, forms. Check daily.
- **First month** — Rankings, conversion trends. Compare to baseline.
- **Ongoing** — Uptime, performance. Set up alerts.

## Optimization

- **Baseline** — Document current metrics. Traffic, conversion, rankings.
- **Backlog** — What to improve? A/B tests, content, SEO. Prioritize.
- **Quarterly** — Review. What's working? What's not? Adjust.
