Content can't rank if the technical foundation is broken. Fix these first.

## Crawlability

- **Robots.txt** — Not blocking important pages. Sitemap referenced.
- **Sitemap** — All important URLs. Submitted to Search Console.
- **Indexation** — No accidental noindex on key pages. No duplicate content issues.

## Speed

- **Core Web Vitals** — LCP, INP, CLS. All under target.
- **Server** — TTFB under 600ms. Hosting and CDN matter.
- **Assets** — Images optimized. JS and CSS minimal. Lazy load below the fold.

## Structure

- **Canonicals** — Every page has one. No self-referencing errors.
- **Schema** — Organization, BreadcrumbList. Service and Article where applicable.
- **Headers** — One H1. Logical H2/H3. No skipped levels.
